Adds `.cleo/adrs/ADR-085-cross-db-invariants.md` cataloguing the five
cross-database reference invariants identified by the Saga T10281
SG-BRAIN-DB-RESILIENCE audit (Epic T10285 E4-DB-CROSS-LINKS):

- **I1** `brain-anchor-task-fk` — `brain_observations.task_id` → `tasks.id`
- **I2** `manifest-blob-owner-fk` — `pipeline_manifest.task_id` /
  `blob_attachments.doc_slug` → `tasks.id`
- **I3** `nexus-project-id-consistency` — `nexus.db.project_id` ↔ canonical
  project identity (base64url-of-cwd / `.cleo/project-context.json`)
- **I4** `llmtxt-session-task-fk` — `llmtxt.db.session_id` ↔
  `tasks.db.sessions.id`
- **I5** `conduit-job-anchor-fk` — `delivery_jobs.payload` anchor →
  `tasks.id` or `brain_observations.id`

Each invariant has a name, description, SQL validation procedure, and
owner-gated repair action on violation. Status `Proposed`; moves to
`Accepted` once T10322 (`cleo doctor db-substrate`) and T10323
(`E_PROJECT_ID_DRIFT` runtime gate) ship.
