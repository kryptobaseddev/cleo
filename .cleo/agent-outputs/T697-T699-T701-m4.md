## Tables Created

### brain_weight_history (T697)
- 12 columns: id, edge_from_id, edge_to_id, edge_type, weight_before, weight_after,
  delta_weight, event_kind, source_plasticity_event_id, retrieval_log_id, reward_signal, changed_at
- 6 indexes: edge composite, from, to, changed_at, event_kind, plasticity_event
- Retention: rolling 90 days (Step 9d sweep, Wave 3 T690)
- Decay events NOT written; only LTP/LTD/Hebbian/prune/external above 1e-6 threshold

### brain_modulators (T699)
- 8 columns: id, modulator_type, valence, magnitude, source_event_id, session_id, description, created_at
- 5 indexes: type, session, created_at, source_event, valence
- Inserted by backfillRewardSignals per task outcome

### brain_consolidation_events (T701)
- 7 columns: id, trigger, session_id, step_results_json, duration_ms, succeeded, started_at
- 3 indexes: started_at, trigger, session
- Required by T628 auto-dream scheduling
