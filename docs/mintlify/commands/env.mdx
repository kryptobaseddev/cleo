# cleo env

Display and validate CLEO environment configuration.

{% hint style="info" %}**Agent Relevance**: Medium - Useful for debugging environment issues{% endhint %}

## Synopsis

```bash
cleo env [OPTIONS]
```

## Description

The `env` command displays the current CLEO environment configuration, including:

- Environment variables
- Configuration file locations
- Storage backend settings
- MCP server configuration
- Active project settings

This is useful for debugging configuration issues and verifying setup.

## Options

| Option | Description |
|--------|-------------|
| `--check` | Validate configuration |
| `--json` | Output in JSON format |
| `--human` | Output in human-readable format |
| `--sensitive` | Include sensitive values (secrets) |

## Examples

{% tabs %}

```bash Show environment
cleo env
```

```bash Validate configuration
cleo env --check
```

```bash JSON output
cleo env --json
```

{% endtabs %}

## Output

```json
{
  "version": "2026.2.5",
  "cwd": "/home/user/project",
  "cleoHome": "/home/user/.cleo",
  "configFile": ".cleo/config.json",
  "storage": {
    "engine": "sqlite",
    "path": ".cleo/cleo.db"
  },
  "mcp": {
    "enabled": true,
    "server": ".cleo/mcp-server.json"
  }
}
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `CLEO_HOME` | CLEO configuration directory |
| `CLEO_CONFIG` | Path to config file |
| `CLEO_STORAGE` | Storage engine: `json` or `sqlite` |
| `CLEO_LOG_LEVEL` | Logging verbosity |

## Exit Codes

| Code | Meaning | Recoverable |
|------|---------|:-----------:|
| 0 | Success | - |
| 8 | Configuration error | Yes |
