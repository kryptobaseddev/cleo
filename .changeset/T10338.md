Adds `scripts/lint-invariant-registry.mjs` — a baseline-gated CI lint that
validates the central invariants registry at `packages/contracts/src/invariants/`
in both directions:

- FORWARD check — for every entry in `INVARIANTS_REGISTRY`, asserts that
  `runtimeGate.module` exists, `runtimeGate.functionName` is exported by
  that module, `lintRule.lintScript` / `doctorAudit.lintScript` paths exist,
  and every `tests[]` entry resolves to an existing file or directory.
- REVERSE check — walks ADR-073 (I1-I8 inline declarations), ADR-070
  (ORC-001..ORC-014 table rows), and ADR-056 (D1-D6 H3 headings); asserts
  each declared code is in the registry OR explicitly noted in
  `DOCS_ONLY_OVERRIDES` with a reason.

Modes: default (baseline-check), `--strict` (zero tolerance), `--baseline`
(regenerate baseline), `--json` (machine-readable output for R6 doctor
audit at T10340). Baseline pinned at 4 forward-check violations against
28 registry entries + 28 ADR-declared codes.
