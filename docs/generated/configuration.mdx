Configuration reference for **cleocode**.

## forge-ts.config.ts

Create a `forge-ts.config.ts` file in your project root:

```typescript
import { defineConfig } from "@forge-ts/core";

export default defineConfig({
  rootDir: ".",
  outDir: "docs/generated",
});
```

## `ConfigFormat`

Supported configuration file formats.  - `"json"` - Standard JSON - `"jsonc"` - JSON with comments (comment-preserving via jsonc-parser) - `"yaml"` - YAML (via js-yaml) - `"toml"` - TOML (via iarna/toml)
