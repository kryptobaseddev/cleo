## Data Files

| File | Purpose |
|------|---------|
| `.cleo/todo.json` | Active tasks |
| `.cleo/todo-archive.json` | Completed tasks |
| `.cleo/config.json` | Configuration |
| `.cleo/todo-log.jsonl` | Audit trail |
| `.cleo/sessions.json` | Session state |

## JSON Output

All commands output JSON when piped:

```bash
cleo list | jq '.tasks[0]'
```

Response envelope:
```json
{
  "$schema": "...",
  "_meta": { "command": "list", "version": "0.60.0" },
  "success": true,
  "tasks": [...]
}
```

<Warning>
Never edit `.cleo/*.json` files directly. Use CLI commands only.
</Warning>
