# cleo mcp-install

Build and configure the CLEO MCP server for AI tool integration

## Synopsis

```bash
cleo mcp-install
```

## Description

Builds the CLEO MCP server (if not already built) and outputs the JSON configuration snippet needed to add it to your AI tool's MCP settings.

Supports Claude Desktop, Claude Code, Cursor, and other MCP-compatible tools.

## Examples

```bash
# Build and get config
cleo mcp-install
```

| Code | Description |
|------|-------------|
