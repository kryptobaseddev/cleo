**@cleocode/lafs** is a TypeScript documentation toolkit that performs a single AST traversal of your project and produces API docs, OpenAPI specs, executable doctests, and AI context files in one pass.

## Features

- Full TypeScript support with TSDoc extraction
- Auto-generated API reference from source code
- Executable `@example` blocks as doctests
- AI-ready context files from a single build pass

## Installation

```bash
npm install -D @cleocode/lafs
```

## Quick Example

```ts
const mapping = getErrorCodeMapping('TaskNotFound');
// { jsonRpcCode: -32001, httpStatus: 404, httpTypeUri: '...', grpcStatus: 'NOT_FOUND', grpcCode: 5 }
```
