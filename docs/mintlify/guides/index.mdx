## Core Workflows

### Daily Task Management
1. Start session: `cleo session start --auto-focus`
2. Check dashboard: `cleo dash`
3. Work on focused task
4. Complete and move: `cleo complete T001 && cleo focus set T002`
5. End session: `cleo session end`
