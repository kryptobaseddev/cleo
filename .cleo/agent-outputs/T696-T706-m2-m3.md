### M2 — brain_plasticity_events expansion (T696)

**Migration**: `packages/core/migrations/drizzle-brain/20260416000002_t673-plasticity-events-expand/migration.sql`

Added 5 observability columns per spec §2.1.2:

| Column | Type | Purpose |
|--------|------|---------|
| `session_id` | TEXT | Session attribution (was in Drizzle, missing from live table) |
| `weight_before` | REAL | Edge weight before event — enables history view |
| `weight_after` | REAL | Edge weight after event (= CLAMP(before + delta_w, 0, 1)) |
| `retrieval_log_id` | INTEGER | Soft FK to brain_retrieval_log.id — causal trace |
| `reward_signal` | REAL | R-STDP reward signal at event time |
| `delta_t_ms` | INTEGER | Δt between the two spikes |

New indexes: `idx_plasticity_retrieval_log`, `idx_plasticity_reward`

**Migration strategy**: CREATE TABLE IF NOT EXISTS with full schema (includes session_id) + ALTER TABLE ADD COLUMN for M2 columns. ensureColumns safety net added for all 6 columns.

### M3 — brain_page_edges plasticity columns (T706)

**Migration**: `packages/core/migrations/drizzle-brain/20260416000003_t673-page-edges-plasticity-columns/migration.sql`

Added 6 plasticity tracking columns per spec §2.1.3:

| Column | Type | Constraint | Purpose |
|--------|------|-----------|---------|
| `last_reinforced_at` | TEXT | nullable | ISO timestamp of last LTP event |
| `reinforcement_count` | INTEGER | NOT NULL DEFAULT 0 | LTP event count lifetime |
| `plasticity_class` | TEXT | NOT NULL DEFAULT 'static' | 'static' \| 'hebbian' \| 'stdp' |
| `last_depressed_at` | TEXT | nullable | ISO timestamp of last LTD event |
| `depression_count` | INTEGER | NOT NULL DEFAULT 0 | LTD event count lifetime |
| `stability_score` | REAL | nullable | tanh(rc/10) × exp(-(days/30)) |

New indexes: `idx_brain_edges_last_reinforced`, `idx_brain_edges_plasticity_class`, `idx_brain_edges_stability`

**Seed**: `UPDATE brain_page_edges SET plasticity_class = 'hebbian' WHERE edge_type = 'co_retrieved'` (idempotent)
