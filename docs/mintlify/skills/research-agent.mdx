# Research Agent Skill

The research agent skill provides comprehensive research capabilities, gathering information from multiple sources and producing structured outputs.

## Overview

<CardGroup cols={2}>
  <Card title="Multi-Source" icon="globe">
    Web search, library docs, Reddit discussions
  </Card>
  <Card title="Structured Output" icon="file-lines">
    Markdown reports with citations
  </Card>
  <Card title="Task Linking" icon="link">
    Connect research to CLEO tasks
  </Card>
  <Card title="Manifest Integration" icon="book">
    Automatic MANIFEST.jsonl entries
  </Card>
</CardGroup>

## Sources

The research agent integrates multiple data sources:

| Source | Provider | Best For |
|--------|----------|----------|
| Web Search | Tavily | Current information, news |
| Library Docs | Context7 | Framework/library documentation |
| Reddit | Tavily | Community discussions, opinions |
| Direct URLs | Tavily Extract | Specific page content |

## Usage

<Tabs>
  <Tab title="General Research">
    ```bash
    cleo research "React 19 new features"
    ```
  </Tab>
  <Tab title="Library Docs">
    ```bash
    cleo research --library react "useEffect cleanup patterns"
    ```
  </Tab>
  <Tab title="Reddit Discussions">
    ```bash
    cleo research --reddit "CLEO task management" --subreddit programming
    ```
  </Tab>
  <Tab title="URL Extraction">
    ```bash
    cleo research --url "https://docs.example.com/guide"
    ```
  </Tab>
</Tabs>

## Research Depth

<CardGroup cols={3}>
  <Card title="Basic" icon="layer-group">
    5-10 sources, quick overview
  </Card>
  <Card title="Standard" icon="layer-group">
    10-15 sources (default)
  </Card>
  <Card title="Deep" icon="layer-group">
    15-25 sources, comprehensive
  </Card>
</CardGroup>

```bash
# Quick research
cleo research "topic" -d basic

# Comprehensive research
cleo research "topic" -d deep
```

## Output Structure

Research outputs follow a consistent structure:

```markdown
# Research: [Topic]

**Date**: 2026-01-20
**Sources**: 15
**Depth**: standard

## Executive Summary
Key findings in 3-5 sentences.

## Key Findings
1. Finding one with citation [1]
2. Finding two with citation [2]
...

## Sources
[1] https://example.com/article
[2] https://docs.library.com/guide
...

## Recommendations
- Actionable item 1
- Actionable item 2
```

## Task Linking

Connect research to CLEO tasks:

```bash
# Link during research
cleo research "auth patterns" --link-task T005

# Link after research
cleo research link T005 research-id-123
```

## Manifest Integration

Research outputs automatically create manifest entries:

```json
{
  "id": "auth-patterns-2026-01-20",
  "file": "agent-outputs/2026-01-20_auth-patterns.md",
  "title": "Authentication Pattern Research",
  "key_findings": [
    "JWT with refresh tokens is current best practice",
    "OAuth 2.0 PKCE recommended for public clients",
    "Session duration: 15 min access, 7 day refresh"
  ],
  "linked_tasks": ["T005"]
}
```

## Subcommands

<AccordionGroup>
  <Accordion title="research init" icon="play">
    Initialize the research outputs directory:
    ```bash
    cleo research init
    ```
    Creates `claudedocs/agent-outputs/` with MANIFEST.jsonl
  </Accordion>

  <Accordion title="research list" icon="list">
    List research entries from manifest:
    ```bash
    cleo research list
    cleo research list --status complete --limit 10
    ```
  </Accordion>

  <Accordion title="research show" icon="eye">
    Show research entry details:
    ```bash
    cleo research show <id>
    cleo research show <id> --full  # Include file content
    ```
  </Accordion>

  <Accordion title="research inject" icon="syringe">
    Get subagent injection template:
    ```bash
    cleo research inject
    cleo research inject --clipboard
    ```
  </Accordion>
</AccordionGroup>

## Subagent Workflow

When using research in orchestrated workflows:

<Steps>
  <Step title="Inject Protocol">
    Use `cleo research inject` to get the research protocol for subagents
  </Step>
  <Step title="Subagent Research">
    Subagent performs research and writes to MANIFEST.jsonl
  </Step>
  <Step title="Query Results">
    Parent agent queries via `cleo research list` and `cleo research show`
  </Step>
  <Step title="Read Summaries">
    Read only `key_findings`, not full research files
  </Step>
</Steps>
