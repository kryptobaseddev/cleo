## Technical Details

### ID Generation (idempotency key)
- RCASD format: `{taskId}-rcasd-{phase}-{slug}`
- Loose format: `{taskId}-loose-{slug}` (or `loose-{slug}` if no task ID)
- Slug generation: lowercase, non-alphanumeric → hyphens
- Example: `T002-rcasd-specification-spec` from `T002/specification/spec.md`

### Schema
- Table: `pipeline_manifest`
- Primary key: `id` (text, unique)
- Duplicate detection: Check `id` before insert
- Fallback: Log errors, increment skipped counter
