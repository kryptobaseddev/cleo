## Deliverables

### 1. Migration SQL

**File**: `packages/core/migrations/drizzle-tasks/20260416000000_t796-attachments/migration.sql`

Two tables added to tasks.db:

- `attachments` — registry row per unique SHA-256; stores serialised `Attachment` JSON + `ref_count`
- `attachment_refs` — junction table (PK: `attachment_id`, `owner_type`, `owner_id`) with FK cascade on `attachments.id`

Indexes:
- `idx_attachments_sha256` (UNIQUE) — O(1) dedup lookup on `put`
- `idx_attachment_refs_owner` — owner fan-out queries (`listByOwner`)
- `idx_attachment_refs_attachment_id` — ref-count audits

### 2. Drizzle Schema

**File**: `packages/core/src/store/tasks-schema.ts` (appended)

Added to the end of the existing schema file:

- `ATTACHMENT_OWNER_TYPES` const — matches `AttachmentRef['ownerType']` from contracts
- `attachments` table — id, sha256 (unique), attachment_json, created_at, ref_count
- `attachmentRefs` table — composite PK + two indexes
- `AttachmentRow`, `NewAttachmentRow`, `AttachmentRefRow`, `NewAttachmentRefRow` type exports

### 3. Store Module

**File**: `packages/core/src/store/attachment-store.ts`

Exports:
- `AttachmentStore` interface — `put`, `get`, `getMetadata`, `listByOwner`, `ref`, `deref`
- `createAttachmentStore()` factory

Key design decisions:
- SHA-256 via Node's built-in `crypto.createHash('sha256')` — no `llmtxt` dep required at this layer
- Storage path: `.cleo/attachments/sha256/<hash[0..2]>/<hash[2..]>.<ext>` (256-bucket prefix sharding)
- `put` is idempotent by SHA-256 — same content = one file, one row, `ref_count++`
- `deref` purges the blob file + row when `ref_count` reaches 0 (best-effort `rm`, never throws on missing file)
- MIME-to-extension map covers 15 common types; fallback `.bin`

### 4. Contracts Index

**File**: `packages/contracts/src/index.ts`

Added exports for `Attachment`, `AttachmentMetadata`, `AttachmentRef`, all five variant interfaces, and all Zod schemas from `attachment.ts` + `attachment-schema.ts`. T795 shipped the files but didn't wire them into the public package surface.
