Closes the structural-bug deferral that ADR-068 §Decision/Known structural
bugs item 1 flagged on 2026-05-23 (T10306). The audit
(`sg-brain-db-resilience-deep-audit-2026-05-23` §1.2) found a nested
subdirectory `$XDG_DATA_HOME/cleo/nexus/` shadowing the canonical flat
global tier with duplicate `nexus.db` (~258 KB) + `signaldock.db`
(~245 KB) + stale `*-pre-cleo.db.bak` sidecars. Neither file is reached by
any live opener; they are migration debris from an incomplete 2026-04-28
flattening.

ADR-086 ratifies the **BAN** disposition: the canonical CLEO global-tier
layout is flat (`<cleoHome>/<role>.db`), the nested subdirectory MUST NOT
exist on post-T10321 installs, and the nested debris has no recoverable
value.

Three deliverables:

1. **`scripts/migrate-nested-nexus.mjs`** — idempotent dry-run-friendly
   migration tool. Allowlist-only deletion (eight specific filenames +
   `cache/` empty-subdir + the nested root itself); refuses to touch any
   unexpected entry as defence-in-depth. Flags: `--dry-run`, `--no-confirm`,
   `--cleo-home <path>`. Exit codes 0 / 1 / 2 / 3 map cleanly to no-op /
   unexpected-only / I/O-error / user-declined.

2. **Runtime warning in `packages/core/src/store/nexus-sqlite.ts`** — new
   `detectAndWarnOnNestedNexus()` helper called from `getNexusDb()` before
   `mkdirSync()`. One-shot per process (gated by an in-memory `Set` keyed
   on the absolute nested path). Non-blocking, non-throwing — the
   canonical flat `nexus.db` open proceeds normally regardless. Emits a
   structured pino warning naming the nested path, the canonical path,
   the migration command, and the ADR / task IDs.

3. **`packages/core/src/store/__tests__/nested-nexus-migration.test.ts`** —
   12 vitest cases covering: (a) no-nested → no-op + no warning;
   (b) warning fires + idempotent one-shot; (c) sentinel-path helper
   returns the expected absolute path; (d–h) migration `plan()` allowlist /
   empty-subdir / defence-in-depth / no-op / two-pass idempotency; (i–l)
   migration `execute()` deletion sweep / unexpected-blocks-root-removal /
   idempotency / flat-tier sibling preservation.

Out of scope (filed as follow-ons under Saga T10281): auto-invocation on
`cleo init` / `cleo upgrade`, `cleo doctor db-substrate` integration,
ADR-068 §Decision/Known structural bugs item 1 amendment to reference
ADR-086.

Saga: T10281. Epic: T10285. ADR: ADR-086.
