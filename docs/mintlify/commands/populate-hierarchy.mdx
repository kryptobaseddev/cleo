# cleo populate-hierarchy

Populate parentId field based on naming convention and depends→epic relationships.

<Info>**Agent Relevance**: Medium - Useful for batch hierarchy organization</Info>

## Synopsis

```bash
cleo populate-hierarchy [OPTIONS]
```

## Description

The `populate-hierarchy` command automatically assigns parent tasks based on two heuristics:

1. **Naming convention**: Tasks with titles like "T001.1" are assigned parentId "T001"
2. **Depends on epic**: If a task depends on a single epic, that epic becomes the parent
