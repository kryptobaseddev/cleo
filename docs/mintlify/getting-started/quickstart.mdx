# Quick Start

Get CLEO installed and managing your first task in under 5 minutes.

## TL;DR - Just Install It

<Tabs>
  <Tab title="npm (Recommended)">
    ```bash
    npm install -g @cleocode/cleo
    ```

    Requires Node.js 24+. Verify with `node --version`.
  </Tab>
  <Tab title="MCP-Only (No Global Install)">
    Add to your AI tool's MCP configuration -- no global install needed:
    ```json
    {
      "mcpServers": {
        "cleo": {
          "command": "npx",
          "args": ["-y", "@cleocode/cleo@latest", "mcp"]
        }
      }
    }
    ```
  </Tab>
  <Tab title="From Source (Contributors)">
    ```bash
    git clone https://github.com/kryptobaseddev/cleo.git
    cd cleo && npm install && npm run build && npm link
    ```
  </Tab>
</Tabs>

## Initialize Your Project

```bash
cd /path/to/your/project
cleo init
```

This creates a `.cleo/` directory with `tasks.db` (SQLite), `config.json`, and schema files. It also auto-injects instructions into `CLAUDE.md`, `AGENTS.md`, and `GEMINI.md`.

## Verify Installation

```bash
cleo version
cleo validate
```

## Your First Task

<CodeGroup>
```bash Create Task
cleo add "Fix login bug" --priority high --labels bug,auth
```

```json Output
{
  "success": true,
  "task": {
    "id": "T001",
    "title": "Fix login bug",
    "status": "pending",
    "priority": "high",
    "labels": ["bug", "auth"]
  }
}
```
</CodeGroup>

## Essential Commands

| Command | Description | Example |
|---------|-------------|---------|
| `cleo add` | Create a task | `cleo add "Task title"` |
| `cleo list` | View tasks | `cleo list --status pending` |
| `cleo complete` | Mark done | `cleo complete T001` |
| `cleo start` | Set active task | `cleo start T001` |
| `cleo dash` | Project overview | `cleo dash` |

## Daily Workflow

```bash
# Morning: Start your session
cleo session start --scope epic:T001 --auto-focus
cleo list --status pending
cleo start T002

# During work: Track progress
cleo update T002 --notes "Fixed validation logic"
cleo complete T002
cleo start T003

# Evening: Clean up
cleo archive
cleo session end --note "Completed auth tasks"
```

<Tip>
Use `ct` as a shorthand alias for `cleo`. Both commands are installed automatically.
</Tip>
