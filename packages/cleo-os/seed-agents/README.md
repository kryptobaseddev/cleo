# CleoOS Seed Agents

Seed agents are bundled with CleoOS and automatically loaded by `AgentRegistry.loadSeedAgents()`.

Place agent definition files here. Each file should export or describe an agent
following the `AgentDefinition` contract in `packages/cleo-os/src/registry/agent-registry.ts`.

This directory is intentionally empty in the initial skeleton (T640). Seed agents will be
added under a follow-up epic.
